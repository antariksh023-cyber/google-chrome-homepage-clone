# Google Chrome Homepage Clone

This project is a responsive clone of the Google Chrome homepage built as part of my Tailwind CSS learning journey.

## 🚀 Technologies Used

- HTML5
- CSS3
- Tailwind CSS

## 📚 Concepts Practiced

- Tailwind CSS Utility Classes
- Flexbox
- Spacing and Alignment
- Typography
- Hover Effects
- Modern UI Development

## 🎯 Purpose

The purpose of this project was to strengthen my understanding of Tailwind CSS by recreating a familiar interface while improving my front-end development skills.
